# Exercise 07: String Uppercase Master

## 📝 Description
In this exercise, I moved from handling single characters to manipulating full strings of text in C. The program takes a complete string from the user and converts all its lowercase letters to uppercase.

## 🛠️ Concepts Learned
- **Strings as Arrays:** Understanding that strings in C are just arrays of characters ending with a null terminator (`\0`).
- **While Loops:** Using a loop to iterate through the string characters one by one.
- **ASCII Math:** Applying the same calculation (`- 32`) to transform characters from lowercase to uppercase.

## 💻 Compilation & Usage
`cc string_upper.c -o app && ./app`
