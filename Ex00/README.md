# Exercise 00: CNOS

## 📝 Description
In this exercise, I implemented a basic C program to display a message on the standard output. 
Following the **1337/42 standards**, I avoided using the `printf` function and instead used the **`write`** system call.

## 🛠️ Concepts Learned
- Using the `unistd.h` library.
- Understanding **File Descriptors** (1 for Standard Output).
- Compiling with `cc` (GCC).

## 💻 How to run
1. Compile the file:
   `cc hello.c -o app`
2. Run the program:
   `./app`
